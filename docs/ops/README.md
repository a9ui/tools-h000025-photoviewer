# Ops Examples

This folder contains copy-time examples for project-local operating notes.
They are examples, not authoritative workspace policy.

Use the workspace docs from the project root for current rules:

- `..\..\System\docs\agmsg.md`
- `..\..\System\docs\oracle.md`
- `..\..\System\docs\workflow.md`
- `..\..\System\docs\workspace-hygiene.md`
- `..\..\System\docs\session-hygiene.md`

When a project enables a lane, copy the matching `.example` file to a local
Markdown file, remove irrelevant sections, and keep durable outcomes in GitHub
issues, PRs, Actions, project docs, and verified reports.
