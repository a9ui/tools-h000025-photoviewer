# PhotoViewer Startup

Use this folder as the project entry point.

1. Read AGENTS.md.
2. Read PROJECT.md, DESIGN.md, and project.toml.
3. For a clean AI/team handoff, read `docs/ai-implementation-brief.md`.
4. For product behavior, read `docs/browser-feature-contract.md` first. It is the Browser product source of truth.
5. For WPF work, then read `docs/wpf-product-spec.md` and `docs/browser-to-wpf-parity-plan.md`.
6. For the current cross-surface audit and remaining priorities, read `docs/product-review-20260718.md`.
7. Check GitHub milestone/issues/PRs.
8. Run powershell -ExecutionPolicy Bypass -File .\scripts\verify-project.ps1.
9. Update SQLite through System/scripts/status.py from the Tools root.

Linear is intentionally not part of this workflow.
