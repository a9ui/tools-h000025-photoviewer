# PhotoViewer Startup

Use this folder as the project entry point.

1. Read AGENTS.md.
2. Read PROJECT.md, DESIGN.md, and project.toml.
3. Check GitHub milestone/issues/PRs.
4. Run powershell -ExecutionPolicy Bypass -File .\scripts\verify-project.ps1.
5. Update SQLite through System/scripts/status.py from the Tools root.

Linear is intentionally not part of this workflow.
